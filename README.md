# Web Scraper - Book Prices

This project is a Python web scraper that extracts book titles and prices from a website and saves them into a CSV file.

## Features
- Extracts book titles and prices
- Saves data into CSV format
- Handles encoding issues (UTF-8)
- Simple and efficient

## Technologies
- Python
- requests
- BeautifulSoup
- pandas
